# hydro-flat

## Synopsis

Flat interface for [hydro](https://github.com/hydrojs/hydro)

```js
suite('My module', function() {
  test('it really works', function() {

  });
});
```

## Usage

```js
hydro.set({
  plugins: ['hydro-flat'],
});
```

## Installation

#### npm:

```bash
npm install hydro-flat
```

#### component:

```bash
component install hydrojs/hydro-flat
```

#### standalone:

```bash
<script src="hydro-flat"></script>
```

## Tests

```bash
$ npm test
```

Coverage:

```bash
$ npm run coverage
```
